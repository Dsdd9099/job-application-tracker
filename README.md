# Job Application Tracker

A full-stack web application that helps users manage and track their job applications efficiently.

## Features
- Add, view, update, and delete job application entries (CRUD functionality)
- Secure backend with Flask and MySQL integration
- Dynamic and responsive frontend using HTML, CSS, and Jinja templates
- RESTful routes to handle form submissions and database interactions

## Tech Stack
- Backend: Python, Flask
- Database: MySQL
- Frontend: HTML, CSS, Jinja2
- Tools: Git, VS Code
